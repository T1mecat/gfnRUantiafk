# gfnRUantiafk
Анти афк для русского клиента GFN.
Просто висит в трее, для завершения просто закрыть выпадающим меню.
Никак не влияет на игровой процесс, пока жфн активно используется не делает вообще ничего, по истечении 13 минут без действий заставляет окно обновиться. 
Если окно свёрнуто, заставит его обновиться раз в 13 минут.
